# 💣 Бомба-Помодоро (Pomodoro Bomb Timer)

Десктопное приложение для управления временем по методу Помодоро, но с высокими ставками! Если вы не успеете выполнить задачу и нажать кнопку «Разминировать» до истечения таймера, бомба взорвется на весь экран.

## ✨ Особенности

* **Визуализация фитиля:** Анимированный фитиль сгорает по мере истечения времени.
* **Настраиваемые интервалы:** Задавайте время на работу и отдых (если время отдыха равно 0, таймер переходит в ручной режим).
* **Поверх всех окон:** Опциональная возможность закрепить таймер поверх других приложений, чтобы бомба всегда была на виду.
* **Статистика сессии:** Подсчет количества успешно разминированных бомб и общего времени сфокусированной работы.
* **Экран провала:** Полноэкранное сообщение "БАБАХ!" при истечении времени.

## 🛠 Требования

* Python 3.x
* Библиотека `tkinter` (обычно поставляется вместе с Python)
* Библиотека `pygame` (опционально, для кастомного звука взрыва)
## 🚀 Установка приложения
1. Скачать из раздела "Releases" `.exe` файл.
2. Запустить.

## 🚀 Установка репозитория

1. Убедитесь, что у вас установлен Python.
2. Клонируйте репозиторий или скачайте файл `bomb_timer.py`.
   
## 🎮 Как использовать

1. Введите название вашей текущей задачи (например, "Написать код").
2. Установите **Время работы** (в минутах).
3. Установите **Время отдыха** (в минутах). Если не хотите, чтобы таймер отдыха запускался автоматически, поставьте `0`.
4. Нажмите **«Поджечь фитиль (Старт)»**.
5. Успейте выполнить задачу и нажать **«Разминировать (Готово!)»** до того, как фитиль догорит!
